## 🚀 Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/YOUR_USERNAME/portfolio.git
   ```
2. Navigate to the project folder:
   ```sh
   cd portfolio
   ```
3. Install dependencies (if applicable):
   ```sh
   npm install
   ```
4. Run the project:
   ```sh
   npm start
   ```
